* Scopri di più sulla virtualizzazione Rocky Linux (KVM)
+
Rocky Linux può fungere da host KVM. Ciò consente di eseguire più macchine virtuali su un singolo server fisico utilizzando la tecnologia Linux Kernel-based Virtual Machine (KVM). L'host KVM non richiede impostazioni di configurazione host esplicite per le LUN ONTAP .
