= Scopri il supporto e le funzionalità ONTAP per gli host Oracle Linux
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Le funzionalità supportate per la configurazione host con NVMe over Fabrics (NVMe-oF) variano in base alla versione di ONTAP e Oracle Linux.

[cols="50,25,25"]
|===
| Caratteristica | Versione host Oracle Linux | Versione ONTAP 


| L'autenticazione in-band sicura è supportata tramite NVMe/TCP tra un host Oracle Linux e un controller ONTAP  a| 
9.4 o successivo
| 9.12.1 o successivo 


| NVMe/TCP è una funzionalità aziendale completamente supportata  a| 
9.0 o successivo
| 9.10.1 o successivo 


| NVMe/TCP fornisce spazi dei nomi utilizzando il nativo `nvme-cli` pacchetto  a| 
8.2 o successivo
| 9.10.1 o successivo 


| Il traffico NVMe e SCSI è supportato sullo stesso host utilizzando NVMe multipath per gli spazi dei nomi NVMe-oF e dm-multipath per le LUN SCSI  a| 
7.7 o successivo
| 9.4 o successivo 
|===

NOTE: NetApp  `sanlun` L'utilità host non è supportata per NVMe-oF. In alternativa, è possibile utilizzare il plug-in NetApp incluso nel pacchetto nativo  `nvme-cli` per tutti i trasporti NVMe-oF.

ONTAP supporta le seguenti funzionalità host SAN indipendentemente dalla versione ONTAP in esecuzione sulla configurazione del sistema.

[cols="75,25"]
|===
| Caratteristica | Versione host Oracle Linux 


| La regola nativa udev in `nvme-cli` il pacchetto fornisce il bilanciamento del carico in base alla profondità della coda per NVMe multipath  a| 
9.6 o successivo



| L'avvio SAN è abilitato tramite il protocollo NVMe/FC  a| 
9.5 o successivo



| Il multipathing NVMe nel kernel per gli spazi dei nomi NVMe è abilitato per impostazione predefinita  a| 
8.3 o successivo



| IL `nvme-cli` il pacchetto include script di connessione automatica che eliminano la necessità di script di terze parti  a| 
8.3 o successivo



| La regola nativa udev in `nvme-cli` il pacchetto fornisce il bilanciamento del carico round-robin per NVMe multipath  a| 
8.3 o successivo

|===

NOTE: Per i dettagli sulle configurazioni supportate, vederelink:https://mysupport.netapp.com/matrix/["Tool di matrice di interoperabilità"^] .
